Web App Development

Hakaniemi Market Hall Electricity

An app that displays Hakaniemi market hall's monthly electricity spending for the year 2019. The data is fetched from an external API to an Express server, from where it's requested by an Angular client app. This project taught me more about creating server apps, and about communication between the server and client side apps, Unit tests were also created for the server app. This work was created as part of an application to Solteq Utilities Academy.

Crypto Analyzer

A crypto currency analyzer created using Angular, where the user can input start and end dates and the app will fetch data from CoinGecko's API and analyze it. This project taught me about fetching and parsing data from an external API. I also got a bit of practice on making unit tests for the application. This app was created as part of an application to Vincit's Rising Star program.

Tic-tac-toe Game

A small tic-tac-toe game created using Angular.

Registration App

An Angular application where the user can log in and create a registration for a party. Utilizes an auth-guard to prevent access to the registration form unless the user is logged in.

Memorization Game

Another small game created using Angular. There are 16 randomized hidden tiles. The player can reveal two of them at a time and if they match they remain visible. The game ends when the player has revealed all tiles.

Student Database

An Express REST API backend app that can add, delete, find and update students and their grades in a mongoDB database. Uses JWT token based authentication.
